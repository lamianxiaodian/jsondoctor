Command Line Tools

jq - The JSON Processor

jq is a powerful command-line JSON processor that lets you slice, filter, map, and transform JSON data.

# Pretty print JSON
cat data.json | jq '.'

# Extract a field
cat data.json | jq '.name'

# Filter array
cat data.json | jq '.users[] | select(.age > 18)'

# Transform data
cat data.json | jq '{name: .name, email: .contact.email}'

json (Python tool)

# Install
pip install json-tool

# Pretty print
python -m json.tool data.json

# Sort keys
python -m json.tool --sort-keys data.json

JavaScript Libraries

Native JSON Object

// Built into modern browsers
JSON.parse(string)
JSON.stringify(object)
JSON.stringify(object, null, 2) // Pretty print

AJV - JSON Schema Validator

const Ajv = require('ajv');
const ajv = new Ajv();

const schema = {
  type: 'object',
  properties: {
    name: {type: 'string'}
  },
  required: ['name']
};

const validate = ajv.compile(schema);
const valid = validate(data);

fast-json-stringify

const fastJson = require('fast-json-stringify');

const stringify = fastJson({
  type: 'object',
  properties: {
    name: {type: 'string'}
  }
});

const json = stringify({name: 'John'});

Python Libraries

Built-in json Module

import json

# Parse JSON
data = json.loads(json_string)

# Generate JSON
json_string = json.dumps(data)
json_string = json.dumps(data, indent=2)  # Pretty print

orjson - Fast JSON Library

import orjson

# Parse JSON
data = orjson.loads(json_string)

# Generate JSON (returns bytes)
json_bytes = orjson.dumps(data)
json_bytes = orjson.dumps(data, option=orjson.OPT_INDENT_2)

jsonschema - JSON Schema Validation

from jsonschema import validate

schema = {
    "type": "object",
    "properties": {
        "name": {"type": "string"}
    },
    "required": ["name"]
}

validate(instance=data, schema=schema)
